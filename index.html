Why SS-Bulma?

Bulma is a great CSS framework, but as front-end developers we're missing some things. Instead of copying the same custom css over and over again, we decided to make a npm module to combat this problem.

The name SS-Bulma is an extension to the naming of Bulma. SS stands for SilverStripe, the CMS we mostly use for Bulma. But SS could also stand for Super Saiyan. All you Dragonball fans know what we mean. :)

Nav

A simple spaced nav element for horizontal and vertical navs. Just put a ul or ol inside a .nav. The li takes both a and span tags.

Horizontal

By default the nav element is horizontal.

Spacings

You can add several spacing classes to increase or decrease spacing.

.is-smallest spacing $nav-gutter-smallest
.is-smaller spacing $nav-gutter-smaller
.is-small spacing $nav-gutter-small
.is-normal spacing $nav-gutter-normal
.is-medium spacing $nav-gutter-medium
.is-large spacing $nav-gutter-large
.is-larger spacing $nav-gutter-larger

Visual

By default a .visual scales with the content. If you want to have default padding, add the .section class to the visual. You can also use the .is-* options to set a default minimal height.
